# Setup

## Run locally

Create and go to the directory where you want to place the repository  

```bash
    cd directory
```
Clone the project

```bash
  git clone https://github.com/from-zero-to-fullstack/html-basics
```

Go to the project directory

```bash
  cd html-basics
```

Open in Visual Studio Code

```bash
  code .
```
